# ![logo](https://github.com/Kostya0Sim/Ore-Excavation-DM/blob/main/pack.png) Ore Excavation DM
This resource pack changes the appearance of the [Ore Excavation](https://www.curseforge.com/minecraft/mc-mods/ore-excavation) GUI to the darker and less saturated tones like in a [Default Dark Mode](https://www.curseforge.com/minecraft/texture-packs/default-dark-mode) resource pack.
